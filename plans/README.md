## Findings considered and rejected

- Literal `oklch(...)` strings in native component styles: rejected because
  React Native 0.81/0.85 cannot be assumed to parse CSS Color 4 syntax on both
  iOS and Android. The supplied OKLCH palette remains canonical, while built-in
  themes use precomputed sRGB hex/rgba values.
- A runtime OKLCH conversion dependency: rejected because the palette is
  static; precomputing the values avoids bundle weight and platform drift.
- Removing the navy/sidebar wrapper hierarchy: rejected because pinned web
  Streamdown intentionally uses sidebar wrappers around background-colored
  table, code, and Mermaid bodies.
